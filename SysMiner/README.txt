※ 메인 소스코드: SysMiner.py

※ 실행 파일: SysMiner.exe

※ 프로그램 사용 방법
 1. 관리자 권한 요청 문구가 뜨면 '예' 버튼 선택하시면 됩니다.
			↓
 2. 기능 선택
   → 좌측 상단 '자동 검사 활성화' 체크 시, 자동으로 윈도우 자동 실행 프로그램에 등록되며, 백그라운드 실행이 됩니다.
      자동 백그라운드 검사가 윈도우가 실행중일 때 수행되며, CPU 검사를 10분마다, 바탕화면 확장자 검사를 2시간마다,
      추출 만료일은 하루에 한 번 검사됩니다.

   → 좌측 하단의 검사 주기 설정(일) 공간은 최종 수정일로부터 설정한 날짜 후에 추출 권장을 위해 알람을 주는 기능입니다.

   → 오른 쪽에는 각 추출 목록이 있으며 필요한 정보만 선택하거나, '전체 선택' 또는 '전체 해제' 버튼을 사용할 수 있습니다.

 3. 추출 목록 중 일부 선택 후 '다음' 버튼 클릭

 4. GUI에 뜨는 로그 확인 후, 바탕화면에 result 폴더 생성 확인

 5. result/Data_Hub에 추출 파일들과 결과 리포트 REPORT.html 파일이 생성됩니다.



!!! 사용시 주의사항 !!!
→ 크롬 혹은 엣지 관련 추출 목록 선택 시, 원활한 추출을 위해 크롬과 엣지가 강제 종료됩니다.
    안내문구가 뜨며, '아니오'버튼을 누르면 추출되지 않고 종료됩니다.

→ '윈도우 업데이트 기록', '방화벽 상태확인', '패스워드 정책', '설치된 소프트웨어'
    위 네개의 목록들은 파워쉘 명령어로 수행되기 때문에, 프로그램 실행 중, 파워쉘이 실행될 수 있습니다.

→ 방화벽 로그는 컴퓨터 내에서 사용자가 방화벽 로깅을 활성화 하여야 추출 할 폴더가 생깁니다.
   따라서 방화벽 로그도 추출하고 싶다면, 아래와 같은 절차를 따라하십시오.
   1) 실행 창에 'Windows Defender 방화벽' 입력 후 실행
   2) 좌측 '고급 설정' 클릭
   3) 좌측 상단에 '로컬 컴퓨터의 고급 보안이 포함된 Windows Defender 방화벽' 마우스 오른쪽 클릭
   4) 속성 클릭 → 공용 프로필 → 로깅의 사용자 지정 클릭
   5) 이름을 수정하지 않고 '확인' → '적용'

→ 사용자의 컴퓨터에 '크롬' 혹은 '엣지'가 설치되어 있지 않으면 추출 목록에서 각 기능을 선택하여도
    추출이 되지 않습니다. 

→ 특정 브라우저가 설치가 되어 있더라도 사용한 캐시가 없더라면 추출되지 않습니다.

→ 자동 검사 활성화 취소를 원한다면,
    체크 버튼을 해제하면 백그라운드 검사를 취소하고 윈도우 실행 프로그램 등록을 해제합니다.

→ '검사 주기 설정(일)' 을 공백으로 두시고 설정을 누르시면 만료일 검사가 해제됩니다.
